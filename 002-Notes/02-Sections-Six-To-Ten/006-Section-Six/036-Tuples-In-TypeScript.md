## Definition

    - Tuple: An array-like structure where each element represents some property of a record.

    - Object Representing a 'drink'
  
      - {  color:      brown  }
      - {  carbonated: true   }
      - {  sugar:      40     }


    - Object Representing a 'drink'

        - [ brown true 40 ] // we lose information the property names.
